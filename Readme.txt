Sản phẩm này gồm 2 phần 
Phần giao diện dùng vue js dữ liệu lấy thông qua api dùng axios
    - Có trang thêm mới Blog 
    - Có trang hiện thị Blog 
    - Có trang sửa Blog
    - Có thể xóa blog
Phần backend dùng laravel có route api resource (index,store,update,destroys), BlogController

Sản phẩm còn thiếu rất nhiều như đăng nhập, đăng ký, phân trang, tìm kiếm.. cũng có thể có sai số sẽ bổ sung thêm sau
